# Music Network Analysis Project

## Purpose
The Music Network Analysis project aims to investigate and visualize the relationships between different music artists and genres using network analysis techniques. The project demonstrates how interconnected various musical elements are and provides insights into trends in music over time.

## Project Structure
- `notebooks/` : Contains Jupyter notebooks that showcase the analysis and visualizations performed.
- `data/` : Includes datasets used for the analysis, containing information about artists, albums, and genres.
- `src/` : Holds the source code and functions used to perform the analysis.
- `requirements.txt` : Lists the dependencies needed to run the project.

## How to Use the Notebooks
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/gtlliu/musicAnalysis.git
   cd musicAnalysis
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Open the Jupyter notebooks in your browser:
   ```bash
   jupyter notebook
   ```
4. Follow the instructions within each notebook to explore the analyses.
